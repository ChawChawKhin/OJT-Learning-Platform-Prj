Conclusion This paper presented the study of the VTP instance gener- ator that automatically generates a new VTP instance from a given Python source code. It finds all the variables in the code using the Python parser, counts the appearing times, selects the variables to be traced that appear three or more times, and makes the corresponding questions, input forms, and correct answers. For evaluations, the application of the proposal to the source codes used in manually generated instances confirmed the conformity of the results. Then, 18 instances on object- oriented programming concepts were newly generated, and assigned to 26 students in Japan, China, and Indonesia where all the instances were well solved basically. In future works, we will continually implement automatic VTP generator for other programming languages such as C, C++, and JavaScript and so on by covering the study of advanced grammar con- cepts, popular libraries.
